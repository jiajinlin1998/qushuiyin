## 项目概述

### 产品定位
一款支持图片和视频去水印的微信小程序，用户无需下载安装，即可快速去除图片/视频中的水印。

### 目标用户
- 需要清理图片水印的普通用户
- 需要处理短视频水印的内容创作者
- 需要批量去水印的运营人员

### 核心指标
- 图片去水印成功率 > 90%
- 视频去水印成功率 > 85%
- 处理响应时间 < 30秒（图片）/ < 60秒（视频）
- 用户留存率 > 30%

## 关键技术点

### 1. Canvas 像素处理

```javascript
// 核心思路
// 1. 获取水印区域像素数据
const imageData = ctx.getImageData(x, y, width, height);
const data = imageData.data;

// 2. 对水印区域进行像素修复（邻域插值）
for (let i = 0; i < data.length; i += 4) {
  // 获取周围像素值，计算平均值填充
  data[i] = averageR;     // R
  data[i + 1] = averageG; // G
  data[i + 2] = averageB; // B
  // Alpha通道保持不变
}

// 3. 将修复后的像素写回Canvas
ctx.putImageData(imageData, x, y);
```

### 2. 视频处理流程

```
用户选择视频 → 播放并框选水印区域 → 上传视频到云存储 → 
调用云函数 → 云函数逐帧提取 → 处理水印 → 合成视频 → 
返回结果 → 用户预览并保存
```

### 3. 云函数接口设计

| 云函数 | 功能 | 输入 | 输出 |
|--------|------|------|------|
| `processVideo` | 视频去水印 | 视频fileID、水印区域坐标 | 处理后视频fileID |
| `batchProcess` | 批量图片处理 | 图片数组、水印区域 | 处理后图片数组 |
| `getHistory` | 获取历史记录 | 用户openid、分页参数 | 历史记录列表 |
| `saveRecord` | 保存处理记录 | 原图、结果图、时间等 | 记录ID |

### 4. 文件管理策略

| 文件类型 | 存储位置 | 保留时间 | 说明 |
|---------|---------|---------|------|
| 用户上传原图 | 云存储 | 30天 | 超期自动清理 |
| 处理结果图 | 云存储 | 30天 | 用户可手动保存 |
| 临时处理文件 | 云存储 | 24小时 | 处理完成后删除 |
| 本地缓存 | 小程序本地 | 7天 | 超期自动清理 |

## 注意事项

### 微信小程序限制
1. **文件大小限制**
   - 小程序包大小限制：主包 2MB，分包 2MB
   - 单文件上传限制：前端 10MB（云函数可调至 100MB）
   - 本地缓存限制：10MB

2. **API 使用限制**
   - `wx.chooseMedia` 支持图片和视频选择
   - `wx.saveImageToPhotosAlbum` 需要用户授权相册权限
   - Canvas 2D API 在基础库 2.9.0+ 支持

3. **云开发限制**
   - 云函数执行超时时间：默认 3 秒，最大 60 秒（**需配置为 120 秒以支持视频处理**）
   - 云存储空间：免费版 5GB，付费版可扩展
   - 云数据库：免费版 2GB 存储，5万次/月读取

### 性能优化建议
1. 图片处理尽量在前端完成，减少云函数调用
2. 视频处理采用分片策略，避免单次处理过大文件
3. 使用 Canvas 离屏渲染提高处理效率
4. 及时清理临时文件和缓存

### 扩展性设计
1. 云函数接口设计保持通用性，便于后续接入 AI 模型
2. 图片处理算法模块化，支持多种算法切换
3. 配置项抽离，便于不同场景快速调整
