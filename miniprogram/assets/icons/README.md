# 去水印小程序 - 图标资源说明

## 需要的图标资源

由于图标资源需要实际的设计文件，以下是需要的图标清单：

### TabBar图标 (miniprogram/assets/icons/)
- home.png / home-active.png (首页图标)
- history.png / history-active.png (历史记录图标)

### 首页功能图标
- image.png (图片去水印)
- video.png (视频去水印)
- select.png (智能框选)
- magic.png (像素修复)
- preview.png (对比预览)
- save.png (保存分享)
- step1.png ~ step4.png (使用步骤图标)

### 其他图标
- empty.png (空状态图标)

## 临时解决方案

在开发测试阶段，可以使用emoji或文字替代图标，确保小程序能正常运行。

## 推荐图标来源
1. 阿里巴巴矢量图标库 (iconfont.cn)
2. IconFont
3. 自定义设计
